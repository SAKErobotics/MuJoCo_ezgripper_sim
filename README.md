# EZGripper Simulation

Clean and organized EZGripper simulation files.

## Directory Structure

- `models/working/` - Working MuJoCo model files
- `meshes/` - STL mesh files for gripper components  
- `scripts/` - Organized scripts
  - `analysis/` - Analysis and debugging scripts
  - `visualization/` - Visualization tools
  - `tests/` - Test scripts
- `docs/` - Documentation and configuration history
- `images/` - Generated images and diagrams

## Key Files

- `models/working/ezgripper_working.xml` - The working baseline MuJoCo model
- `meshes/` - All STL mesh files for gripper geometry

## Usage

Use the working model in `models/working/ezgripper_working.xml` as the reference implementation.
